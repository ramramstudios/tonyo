How to View TABrite Tablatures

Download, install, and open the free TABrite viewer to play and print any .bjo tab file.

⬇ Download TABrite

● Latest: v4.3 Free Windows only No longer updated

TABrite 4.3 is the final release of this software. It is completely free and still the standard viewer for .bjo files. Download the full installer package from the official Bluegrass Banjo site.

⚠️ Mac users: The original Mac version (.sit) is not compatible with modern macOS. See the modern alternatives below — TablEdit can open .bjo files on both Mac and Windows.
🖥 Download TABrite for PC (.zip) 🔗 Official Download Page

🔧 Installation — Windows

  1. 1
    Download and unzip
    Download the ZIP above and extract it. Right-click the ZIP → Extract All, or use 7-Zip / WinRAR.
  2. 2
    Run the installer
    Double-click the installer inside the extracted folder. TABrite will install into C:\Program Files\TABwin43\.
  3. 3
    Install the font — required
    Find Tab.ttf in the TABwin43 folder. Right-click it and choose Install. Without this font, tabs will not display correctly.
  4. 4
    Create a desktop shortcut (optional)
    Right-click TABwin43.exe → Send to → Desktop (create shortcut).
  5. 5
    Check DirectX
    TABrite requires DirectX. On Windows 10/11 this is already installed. If you see graphics errors, run Windows Update or install the DirectX End-User Runtime.
ℹ️ On Windows 10 or 11, if a SmartScreen warning appears when running the installer, click More info → Run anyway. TABrite predates modern code-signing but is safe.

🎵 Opening a Tab File

  1. 1
    Download a tab from the tabs page
    Each song on the tablatures page links to a .zip file. Download and extract it — inside you'll find a .bjo file.
  2. 2
    Open TABrite
    Launch TABrite from your desktop shortcut or Start menu.
  3. 3
    Open the file
    Go to File → Open and navigate to the extracted .bjo file. The tablature will display immediately.
  4. 4
    Play the MIDI (optional)
    Many tabs include a separate .mid MIDI file you can play alongside the tab to hear the melody or the missing harmony part.
💡 Use File → Print inside TABrite to print a clean copy of any tablature. The Tab.ttf font ensures the banjo notation prints correctly.
